Portfolio

Tracking Covid-19

This group project utilized various APIs to create a non-partisan information dashboard for individuals with dynamically updated COVID-19 data in the United States. There is a dropdown function allowing the user to select their state and see current COVID-19 data. There is also a live count of cases, deaths and recoveries at the bottom of the page. Additionally, the site has useful information including informational videos and current CDC recommendations. Click the image to see the deployed website.

Github repository: https://github.com/jordanmorse/trackingcovid

Interactive Day Planner

This project utilizes HTML, CSS, Javascript and JQuery to create a dynamic and interactive dayplanner. A scheduler with hour blocks ranging from 9 am to 5 pm allows you to input tasks and assignments that will be saved even if the webpage is closed. The time blocks are color coded based on if it is past, future or the present time. Click the image to see the deployed website.

Github repository: https://github.com/jordanmorse/dayplanner

Current Weather Dashboard

This weather dashboard uses Javascript, JQuery and OpenWeather APIs to display current weather data as well as a five day forecast with icons. The search function allows a user to input a city of their choice and then saves recent searches as buttons that can be clicked to pull up city data. Click the image to see the deployed website.

Github repository: https://github.com/jordanmorse/weatherdashboard
